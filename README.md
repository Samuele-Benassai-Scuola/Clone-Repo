## GIT RESET

Il comando si usa per spostare la HEAD ad un commit arbitrario, dunque rendendo possibile spostare la HEAD ad un commit passato.

## GIT REVERT

Il comando si usa per creare un nuovo commit cancellando tutte le modifiche avvenute nel commit passato specificato.
